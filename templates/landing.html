Kenapa ZiRa?

Semua fitur yang kamu butuhkan untuk mengatur keuangan pribadi, dikemas simpel dan mudah dipakai.

Dashboard & Laporan

Ringkasan saldo real-time, grafik pemasukan vs pengeluaran, laporan bulanan per kategori.

Multi Dompet

Lihat, edit, dan hapus transaksi per bulan. Filter berdasarkan kategori dan tanggal.

Dari Awal Sampai Pantau

Hanya 3 langkah untuk mulai mencatat keuangan kamu.

1

Daftar

Buat akun dengan Google atau username biasa. Gratis, tanpa verifikasi email.

2

Tambah Dompet

Buat dompet virtual — BCA Tabungan, GoPay, atau uang tunai. Saldo otomatis terhitung.

3

Catat Transaksi

Input pemasukan & pengeluaran harian. Lihat grafik dan laporan bulanan langsung.

Keamanan Data Kamu

Kami serius menjaga privasi dan keamanan data keuangan kamu.

Password Aman

Password di-hash dengan bcrypt — tidak ada yang bisa membaca password asli kamu, termasuk kami.

Session Terenkripsi

Cookie session dibuat dengan crypto/rand, dikirim hanya melalui HTTPS (Secure + HttpOnly).

Data Milik Kamu

Semua data disimpan di server sendiri, tidak dibagi ke pihak ketiga, tidak ada analytics tracker.

Pertanyaan Umum

Beberapa hal yang mungkin kamu tanyakan.

Apakah aplikasi ini gratis?

Ya, 100% gratis. Tidak ada biaya berlangganan, tidak ada iklan. Dibangun untuk penggunaan pribadi.

Data saya disimpan di mana?

Disimpan di database SQLite di server. Tidak ada data yang dikirim ke pihak ketiga. Kamu bisa request export data kapan saja.

Kenapa pakai Google login? Aman?

Google hanya memberikan nama & email kamu — kami tidak bisa mengakses data Google kamu. Password dikelola Google, kamu tidak perlu ingat password baru.

Dibangun dengan teknologi apa?

Backend dengan Go (net/http), frontend Bootstrap 5 + ApexCharts, database SQLite dengan WAL mode. Dijalankan di server Linux.
